# simpleDashboard

A simple login page and dashboard built using typescript, react hooks, axios and Material UI by fetching data from dummy APIs

## Installation

Install necessary libraries

```bash
  npm install
```

## Run

```bash
  npm start
```

The project will run on http://localhost:3000
